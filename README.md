# Tennis Prediction System

Auditierbares Tennis-Einzelspiel-Prognosesystem.

## Ziel

Das System soll echte Tennis-Matches analysieren, Datenquellen prüfen, Evidence speichern, Prognosen vor Matchbeginn einfrieren und Ergebnisse nachträglich auswerten.

## Aktueller Stand

Projektfundament mit FastAPI, Docker und PostgreSQL.
